# Smart AI Password Strength Analyzer

A cybersecurity-focused C project that analyzes password strength using:
- entropy calculation
- rule-based scoring
- attack simulation
- AI-based classification

## Features
- Password strength analysis
- Entropy calculation
- Security recommendations
- Dictionary attack simulation
- Brute force estimation
- Modern terminal UI

## Technologies Used
- C
- File Handling
- ANSI Terminal Colors
- Machine Learning Concepts

## Build

```bash
make
./password_analyzer
